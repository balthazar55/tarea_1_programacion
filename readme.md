# seccion didactica 1 de progrmacion: actividad/formulario de animal favorito usando typscript y nodejs

Estructura:
- `seccion_didactica1_tarea1/` - Proyecto principal con código TypeScript, vistas EJS y scripts.

Breve descripción de archivos importantes
- `server.ts`: es el archivo encargado de controlar el punto de entrada para el servidor de ejecucion, su configuracion y rutas.
- `views`: aca se alamacenan los archivos de las dos vistas requeridas para esta actividad, el formulario de entrada y la vista de salida
- `public/styles.css`: estilos globales del front-end.
- `public/scripts.js`: JavaScript para mejorar la funcionalidad de la pagina
- `tsconfig.json`: configuración de TypeScript.
- `package.json`: scripts y dependencias.


Comentarios en el código:
- He añadido comentarios explicativos en los archivos principales:
	- `src/server.ts`: Explicación de rutas, middleware y puntos de extensión.
	- `views/form.ejs` y `views/display.ejs`: Comentarios en EJS sobre variables (animal/otherAnimal) y cómo extender las vistas.
	- `public/styles.css` y `public/scripts.js`: Comentarios sobre estilos y comportamiento del frontend.
